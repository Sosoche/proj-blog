# Приложение Блога 🚀

Простое, но мощное фулстек-приложение блога, созданное с использованием Spring Boot, React и PostgreSQL.

## 🏗 Архитектура
- **Фронтенд**: React + Vite (Порт 3000)
- **Бэкенд**: Spring Boot + JPA + JWT (Порт 8080)
- **База данных**: PostgreSQL (Порт 5432)
- **Оркестрация**: Docker Compose

## 🛠 Быстрый старт

### Требования
- Docker и Docker Compose
- Make (необязательно, но рекомендуется)

### Запуск приложения
Самый простой способ запустить проект — использовать предоставленный `Makefile`:

```bash
# Сборка образов
make build

# Запуск приложения
make start
```

После запуска:
- 🌐 **Фронтенд**: [http://localhost:3000](http://localhost:3000)
- 🔌 **Бэкенд API**: [http://localhost:8080/api](http://localhost:8080/api)

## 📜 Команды
| Команда | Описание |
|---------|-------------|
| `make start` | Запустить все сервисы в фоновом режиме |
| `make stop` | Остановить все сервисы |
| `make build` | Собрать/пересобрать Docker-образы |
| `make restart`| Перезапустить все сервисы |
| `make logs` | Посмотреть логи в реальном времени |
| `make clean` | Удалить контейнеры и тома (volumes) базы данных |

## 📂 Структура проекта
- `/backend` - Приложение на Spring Boot
- `/frontend` - Приложение на React/Vite
- `docker-compose.yml` - Конфигурация инфраструктуры
